Der Unterschied zwischen unsigned int und unsigned long liegt in der Unterschiedlichen Implementierung bei 64-bit und 32-bit Systemen. So hat ein unsigned long in RV32 eine Länge von 4 Byte und in RV64 eine Länge von 8 Byte. Der unsigned int hat in beiden Fällen eine Länge von 4 Byte. In RV64 werden jedoch auch alle 4-Byte Datentypen in ein 64-Bit-Register geschrieben. Hierbei werden die restlichen, nicht belegten Bits mit Vorzeichen-Bits aufgefüllt, das heißt, ein unsigned Integer hat bei einem 64-Bit System vier Byte voller Vorzeichen-Bits und anschließend die Daten-Bits, während ein unsigned long zur Gänze aus Daten-Bits besteht.
